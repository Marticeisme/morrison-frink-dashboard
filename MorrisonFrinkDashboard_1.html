Setting Up Shared Sync

To share data between your device and your partner's, the easiest free option is a shared Google Sheet. Here's how:

  1. Create a Google Sheet and share it with both of you (Edit access)
  2. Copy the Sheet ID from the URL (the long string between /d/ and /edit)
  3. Paste it below and save

Get a free API key at console.cloud.google.com → Enable Sheets API → Credentials
